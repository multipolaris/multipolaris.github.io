# Brief Projet

## Identité du Projet
**Nom** : Page de dons centralisée pour chaînes YouTube
**Créateur** : Houssen Moshinaly
**Localisation** : Antananarivo, Madagascar

## Mission Principale
Créer une page web hébergée sur GitHub Pages qui promeut et centralise les soutiens financiers pour deux chaînes YouTube :
- **Multipolaris** : Analyses géopolitiques et militaires
- **Houssen Moshinaly** : Technologie et Intelligence Artificielle

## Objectifs Fondamentaux
1. **Conversion** : Maximiser les donations et abonnements Patreon
2. **Centralisation** : Regrouper toutes les méthodes de paiement en une page
3. **Accessibilité** : Faciliter les paiements depuis l'international
4. **Professionnalisme** : Refléter la qualité des contenus produits
5. **Légèreté** : Rester optimisé pour GitHub Pages (pas de dépendances lourdes)

## Contraintes Techniques
- Hébergement statique sur GitHub Pages
- Pas de backend/serveur (HTML/CSS/JS uniquement)
- Performance optimale (chargement rapide)
- Compatible mobile et desktop
- Accessibilité WCAG AA+
- Support des thèmes dynamiques (Neutre, Multi, Tech)

## Méthodes de Paiement
- **Patreon** (priorité : abonnements mensuels)
- **Western Union** (transferts internationaux)
- **Mobile Money** (Telma Mvola)
- **Cryptomonnaies** (USDT TRC-20, Bitcoin)

## Audience
- Abonnés des chaînes YouTube
- Spectateurs souhaitant soutenir la création de contenu indépendant
- Public international (France, Europe, Afrique)

## Contexte Spécifique
Les obstacles géographiques (Madagascar) nécessitent des solutions de paiement flexibles et fiables. La page doit rassurer sur la confiance et la facilité des transactions.
